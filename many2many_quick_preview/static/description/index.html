App Overview

The Many2Many Quick Preview app enhances your Odoo experience by allowing users to instantly preview related records from many2many fields directly within the many2many tags. This app streamlines navigation and boosts productivity by providing quick access to related records, eliminating the need to open them individually.

Key Features:

  • Instantly preview related records in many2many fields without opening them
  • Quick access to detailed record information directly from the many2many tags
  • Boost productivity by reducing time spent switching between records
  • Improves user experience with faster, more efficient navigation
  • Seamlessly integrates into existing Odoo many2many fields and views
